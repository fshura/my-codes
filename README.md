# Телефонная книга

Консольное приложение для управления контактами на C++.

## Возможности

- Добавление контактов
- Просмотр всех контактов
- Удаление контактов
- Сортировка по алфавиту
- Импорт из файла
- Экспорт в файл

## Сборка

### Через CMake

```bash
mkdir build && cd build
cmake ..
make
./phonebook
